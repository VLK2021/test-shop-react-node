# Test-Shop React Node

Цей проєкт складається з двох частин: Backend та Frontend.

## Backend

Для написання серверної частини використовуються наступні технології:

- **Node.js**
- **Express**
- **TypeScript**
- **MySQL2**
- **TypeORM**
- **ESLint**
